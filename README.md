Project Setup
===

```
git clone https://github.com/hashgraph/hedera-cli
cp .env.sample .env
# update .env with node id, node address, 
# operator's id and private key
```

Generating the Executable
===

```
./scripts/build.sh
```

Get Started
===

```bash
./hedera
```

Crypto Transfer
===

```bash
cryptotransfer 1001 44
```
